
📦 Οδηγίες Εγκατάστασης για το Railway

1. Κάνε upload τα αρχεία του zip στο GitHub (νέο repo ή παλιό)
2. Σύνδεσε το GitHub repo στο Railway
3. Railway -> New Project -> Deploy from GitHub
4. Βεβαιώσου ότι το αρχείο 'railway.json' υπάρχει
5. Το bot θα ξεκινήσει μόνο του. Κάθε 60 δευτερόλεπτα σκανάρει:
   - https://inforadar.live/#/dashboard/soccer/live
   - https://inforadar.live/#/dashboard/basketball/live
6. Αν βρει Alg.1 > +0.1 ή < -0.1, στέλνει μήνυμα στο Telegram
7. Δεν χρειάζεται να έχεις ανοιχτό τον υπολογιστή
